<h2>Quick start</h2>
<ul>
  <li>Clone the repo: <code>git clone https://github.com/themewagon/aurora-free.git</code></li>
  <li>Recommended: <strong>Node.js v20.x</strong></li>
  <li><strong>Install:</strong> <code>npm i</code> or <code>pnpm i</code></li>
  <li><strong>Start:</strong> <code>npm run dev</code> or <code>pnpm dev</code></li>
  <li><strong>Build:</strong> <code>npm run build</code> or <code>pnpm build</code></li>
  <li>Open browser: <code>http://localhost:5005</code></li>
</ul>
